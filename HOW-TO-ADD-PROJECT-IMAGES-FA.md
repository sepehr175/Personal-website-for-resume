## 📐 توصیه‌های فنی:

### ابعاد عکس:
- **بهترین نسبت**: 4:3 (مثلاً 1200x900، 800x600)
- **حداقل عرض**: 600px
- **حداکثر عرض**: 1200px (برای سرعت بهتر)

### حجم فایل:
- **هدف**: زیر 200KB برای هر عکس
- **حداکثر**: 500KB
- **فشرده‌سازی**: از [TinyPNG](https://tinypng.com) استفاده کن

### فرمت:
- **ترجیح اول**: JPG (برای عکس‌های واقعی)
- **ترجیح دوم**: PNG (اگه شفافیت لازم داری)
- **WebP**: اگه مرورگرهای قدیمی مهم نیستن

## 🚀 مراحل اضافه کردن:

### مرحله 1: آماده کردن عکس‌ها
```bash
# اگه عکس‌ها خیلی بزرگن، سایزشون رو کوچیک کن
# می‌تونی از این سایت‌ها استفاده کنی:
# - https://tinypng.com (فشرده‌سازی)
# - https://imageresizer.com (تغییر سایز)
```

### مرحله 2: کپی عکس‌ها
```bash
# روش 1: کپی دستی
# فایل‌ها رو از فولدر دانلود کپی کن به:
sepehr-portfolio/frontend/public/projects/

# روش 2: از PowerShell
cd sepehr-portfolio/frontend/public
mkdir projects
# بعد عکس‌ها رو کپی کن
```

### مرحله 3: تست محلی
```bash
cd sepehr-portfolio/frontend
npm start
```

بعد برو به `http://localhost:3000` و بخش Projects رو چک کن.

### مرحله 4: Commit و Push
```bash
cd sepehr-portfolio
git add frontend/public/projects/
git commit -m "Add project screenshots"
git push origin main
```

## 🎨 بهینه‌سازی عکس‌ها:

### اگه عکس‌ها بزرگن:

**روش 1: استفاده از TinyPNG**
1. برو به [tinypng.com](https://tinypng.com)
2. عکس‌ها رو آپلود کن
3. فایل‌های فشرده شده رو دانلود کن

**روش 2: استفاده از ImageMagick (اگه نصب داری)**
```bash
# تغییر سایز به عرض 800px
magick convert original.jpg -resize 800x project.jpg

# فشرده‌سازی
magick convert original.jpg -quality 85 project.jpg
```

**روش 3: استفاده از Photoshop**
- File → Export → Save for Web
- Quality: 80-85
- Width: 800-1200px

## 🐛 عیب‌یابی:

### عکس نمایش داده نمیشه:
1. چک کن اسم فایل درست باشه (حروف کوچیک!)
2. چک کن مسیر درست باشه: `frontend/public/projects/`
3. مرورگر رو Refresh کن (Ctrl+Shift+R)
4. چک کن عکس corrupt نباشه

### عکس خیلی کند load میشه:
1. سایز عکس رو کوچیک‌تر کن
2. از TinyPNG استفاده کن
3. Quality رو کم کن (85 به جای 100)
