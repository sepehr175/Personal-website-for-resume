```

**راه حل:** Backend از SOCKS5 استفاده نمی‌کنه

# اگه ندیدی:
# 1. Backend رو ببند (Ctrl+C)
# 2. دوباره اجرا کن:
npm start
```

```bash
cd backend

npm start
```
